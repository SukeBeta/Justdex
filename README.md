# Justdex

A modern Pokédex web app covering all 1,025 Pokémon (Gen I–IX).

**[Live Demo → sukebeta.github.io/Justdex](https://sukebeta.github.io/Justdex/)**

## Features

- **Pokédex** — Browse, search, filter by type/generation, sort by number or name
- **Detail View** — Base stats, abilities, evolution chains, type-tinted hero
- **Team Builder** — Build a team of 6, type coverage analysis, export shareable card
- **Compare** — Side-by-side stat comparison with radar chart overlay
- **Multilingual** — English, 日本語, 中文, 한국어, Français, Deutsch
- **Responsive** — Desktop and mobile layouts

## Tech Stack

React 19 · TypeScript · Vite · Tailwind CSS · TanStack Query · PokeAPI v2

## Development

```bash
npm install
npm run dev          # Start dev server
npm run build        # Production build
npm run build-data   # Regenerate Pokémon list from PokeAPI
npm run build-names  # Regenerate localized names from PokeAPI
```
